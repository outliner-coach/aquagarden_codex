# PLAN.md

## 개요

이 문서는 앞으로 해야 할 작업을 관리한다.
새 작업을 시작하는 에이전트는 반드시 이 문서를 읽고 우선순위와 상태를 확인한 뒤 작업을 이어간다.

상태 값은 아래 네 가지만 사용한다.

- `todo`
- `in_progress`
- `blocked`
- `done`
