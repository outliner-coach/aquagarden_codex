# 2026-03-05 Improvement Roadmap

조사 기반 개선 계획 문서.
기준 문서:
- `2026-03-05-aquascape-aesthetics.md`
- `2026-03-05-rendering-and-experience.md`

## 목표

현재 앱을 `기능 시연용 3D 수족관`에서 `관상용 디지털 오브제` 수준으로 끌어올린다.

핵심 방향:
- 풍경이 먼저 보이고, 기능은 뒤로 빠진다.
- 물고기보다 구도와 빛이 먼저 읽힌다.
- 화려함보다 차분함과 깊이감이 우선이다.

## 단계별 계획

### Phase 1. 구도와 색 재정리
목표: 첫 화면만 봐도 더 정돈되고 깊어 보이게 만든다.

작업:
- 하드스케이프를 `concave` 또는 `triangular` 구도로 재배치
- 전경 여백과 모래길 확보
- 후경 경사 강화
- 붉은 수초를 한 포컬 포인트로 축소
- 물고기 채도와 크기, 속도 조정

완료 기준:
- 메인 포컬 포인트가 명확하다.
- 전경/중경/후경이 시각적으로 구분된다.
- 물고기가 풍경을 압도하지 않는다.

### Phase 2. 조명과 분위기 모드 강화
목표: 같은 레이아웃도 시간대와 감정이 달라 보이게 만든다.

작업:
- 메인 조명 방향 재설계
- 암부와 하이라이트 대비 강화
- `차분한 아침`, `중성 낮`, `몽환적 저녁` 같은 프리셋 추가
- 필요 시 약한 bloom 도입

완료 기준:
- 최소 3개 분위기 프리셋이 제공된다.
- 조명만 바꿔도 감상 경험이 분명히 달라진다.

### Phase 3. 식재 연결감과 움직임 자연화
목표: 장면의 경계가 덜 인공적으로 보이게 만든다.

작업:
- 유목/바위 연결부에 활착수초 느낌 추가
- 군락 내부 색과 높이의 반복 패턴 완화
- 수초 흔들림 위상 다양화
- 물고기 유영 패턴 완급 조정

완료 기준:
- 수초 군락이 덩어리감은 있으되 기계적으로 보이지 않는다.
- 유목과 바위가 식재로 자연스럽게 연결된다.

### Phase 4. 렌더링 품질과 성능 보강
목표: 미감을 유지한 채 물/유리/하이라이트 품질을 올린다.

작업:
- 색 관리 검토
- `EffectComposer + OutputPass` 기반 후처리 파이프라인 검토
- 약한 bloom 적용 여부 실험
- 반복 오브젝트 instancing 검토
- 확장 시 LOD 전략 수립

완료 기준:
- 콘솔 에러 없이 렌더 품질이 향상된다.
- 프레임 저하 없이 시각 효과가 유지된다.

## 추천 실행 순서

1. Phase 1 전체
2. Phase 2 중 조명 프리셋
3. Phase 3 일부
4. 필요하면 Phase 4

## 현재 바로 착수할 작업

- `AQ-101` 아쿠아스케이프 구도 재정리
- `AQ-102` 색 팔레트와 오브젝트 존재감 조정
- `AQ-103` 조명과 분위기 프리셋 강화
